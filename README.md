* ## Project Overview

This project applies Unsupervised Machine Learning to segment the restaurant market in Gachibowli, Hyderabad. By analyzing features such as cost, cuisine variety, and customer ratings, the model identifies distinct business segments to help stakeholders understand market dynamics and customer preferences.

* ## Tech Stack

Language: Python 3.x

Data Handling: Pandas, NumPy

Machine Learning: Scikit-Learn (Agglomerative Clustering)

Mathematical Analysis: SciPy (Linkage & Dendrogram)

Visualization: Matplotlib, Seaborn

* ## How to Run
Clone this repository.

Install dependencies: pip install pandas scikit-learn scipy matplotlib seaborn

Run the notebook: jupyter notebook ZomatoML.ipynb
